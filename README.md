# test

### Запуск

Ставим необходимые пакеты:
```
npm i
```

Запускаем сборку:
```
npm run dev
```

Приложение откроется по адресу http://localhost:3000/

### Сервер

 * Написан на express.js

### API

 * Адрес: http://localhost:3000/feed?page=[INDEX]
 * Возвращает массив до 10 объектов
 * Обект имеет вид `{"author": string,"description": string,"tags": string,"url": string}`
 * URL видео имеет вид `/v/14528021-425148686832983381`, скрывает расположение и расширение видео.

 ### Фронт

 * Используется виртуальный скролл, в DOM единовременно только 3 видео: текущее, предыдущее и следующее.
 * При запуске приложения запрашивается массив первых 10 видео.
 * Следующие видео запрашиваются при достижении восьмого загруженного видео.
 * При достижении конца ленты появляется соответствующее сообщение.
 * Загруженные страницы кэшируются, чтобы избежать повторных запросов к серверу.
